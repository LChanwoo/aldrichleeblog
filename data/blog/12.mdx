# vercel로 블로그 배포하는 방법

이 블로그는 vercel로 배포되었습니다.
앞으로 팀 프로젝트를 하며 만들 포트폴리오를 관리하기 위한 팀 블로그를 효과적으로 만들기 위해 이 포스트를 작성합니다.


## vercel이란?

vercel은 서버리스 플랫폼으로, 무료로 사용할 수 있습니다.

## 준비물

1. github 계정
2. vercel 계정
3. nextjs, react, javascript, typescript, nodejs에 대한 기본 지식
4. 컴퓨터에 nodejs가 설치되어 있어야 합니다.
https://nodejs.org/en

## vercel로 블로그 배포하는 방법

### 1. vercel에 가입합니다.
<Image alt="" src="/static/images/mkblog/1.png" width={800} height={800} />

https://vercel.com/

vercel은 깃허브 아이디로 가입할 수 있습니다.

### 2. 프로젝트 생성하기

vercel에 로그인하면 아래와 같은 화면이 나옵니다.
<Image alt="" src="/static/images/mkblog/2.png" width={800} height={800} />

Add New > Project를 클릭합니다.

<Image alt="" src="/static/images/mkblog/3.png" width={800} height={800} />

화면을 아래로 내리면 Clone Template이 있습니다.
여기서 Brows All Templetes를 클릭합니다.


<Image alt="" src="/static/images/mkblog/4.png" width={800} height={800} />

tailwind css blog를 검색하고 클릭해줍니다.


<Image alt="" src="/static/images/mkblog/5.png" width={800} height={800} />

Deploy 버튼을 클릭합니다.


<Image alt="" src="/static/images/mkblog/6.png" width={800} height={800} />


이제 거의 다 왔습니다.
사용할 블로그의 repository이름을 입력하고 create를 클릭합니다

<Image alt="" src="/static/images/mkblog/7.png" width={800} height={800} />

배포가 시작되었습니다.
기다려줍니다.


<Image alt="" src="/static/images/mkblog/9.png" width={800} height={800} />

배포가 완료되면 위와 같은 화면이 나옵니다.
continue to Dashboard를 클릭합니다.



<Image alt="" src="/static/images/mkblog/10.png" width={800} height={800} />

visit 버튼을 클릭하면 블로그가 나옵니다.
