- 리눅스 시험을 보았습니다. 굉장히 쉬운 문제들로
  구성되어 있었습니다. 그러나 한 문제를 틀려버리고 말았습니다. 아침에 강의가 시작하기 직전 제 자리
  주변에 있던 3명의 동기들이 리눅스의 구성 요소는 “커널”, “쉘”, “디렉토리”라고 하는 것을 들었습니다.
  그 말을 듣고서 저는 “어? 아닌데? 디렉토리가 아니라 사용자 프로그램 또는 유틸리티라고 부르는 그거
  아님…?”이라고 말했습니다. 그런데 삼인성호(三人成虎)라는 말이 있죠… **사람 셋이 모이면 없는 호랑이도
  만들어 낸다.** 이게 무슨 운명의 장난인지, 시험 문제에 “뙇!”하고 아래와 같이 나왔습니다.

  <aside>

    쉘(Shell) 4. 디렉토리(Directory)

  어…? 커널, 쉘, 디렉토리…? 아침에 들었던 그 순서대로 나오는데, 도저히 참을 수가 없었습니다. 저의 이성은
  1번을 찍으라고 아우성쳤지만, 저의 감성은 “이건 하늘이 정해준 운명이다. 4번을 찍을 수 밖에 없잖아?!!!!”라며
  저의 행동을 장악해버렸습니다. 그래서 어쩔 수 없이 4번을 찍게 되었고 당연하게도 한 문제를 틀리게 되었습니다.
  사실 이 문제는 SQLD를 앞두고 데이터베이스 공부를 하느라 리눅스 공부를 소홀히 하여 완벽하게 리눅스 구성
  요소에 대해 숙지하지 못해서 생긴 일입니다. 완벽하게 알지 못하니 줏대없이 “팔랑귀”해버린 것이지요. 공부를
  할 때는 그것에 대해 완벽하게 알고 넘어가야 한다는 너무나 당연한 사실을 이제서야 알게 되었습니다. 앞으로는
  좀 더 조심하며 공부해야 하겠습니다. 그런데 한 가지 신기한 점이 있습니다. 제가 저 문제를 틀리고 있는
  동안 저를 팔랑귀로 만든 3인 중 한 명인 전상언님이 제 말을 듣고 4번을 찍어서 저 문제를 맞았다는 것입니다.
  이거 살짝 억울하기까지 합니다. 그래도 말 한마디로 사람을 구한다는 말도 있는데, 제가 이번에 말 한마디로
  사람을 구했으니 그것 만은 보람으로 삼는 게 옳겠지요. 생각이 깊어지는 한 주 입니다. … 라고 생각했는데
  1번이 리눅스의 구성요소가 아니였고, 따라서 1번이 답이였습니다. … 그렇습니다. 저 때문에 상언님이 속았군요…
  저는 제가 피해자인줄 알았는데 알고보니 가해자였습니다. 생각이 매우 깊어집니다. 그래도 논란의 여지가
  있어서 1번, 4번 둘 다 정답으로 인정해준다고 하셨습니다. 다행입니다. ㅋㅎ 그럼 제가 진짜로 틀린 문제는
  무엇일까요? 찾아보니

  이 문제를 틀린 듯 합니다. -exec의 경우 물어보지 않고 한번에 다 실행하는 듯합니다. 사용자에게 실행여부를 물어보는 옵션은 -ok 옵션이라고 합니다. 살짝 아쉽습니다.

- SQLD 시험을 보았습니다. 이 시험을 보기 위해 공부하느라 얼마나 힘들었던지… 정말이지 힘들었습니다. 오랜만에 단순히 외워서 푸는 것이 아닌, 외운 것을 가지고 응용해서 푸는 진짜 시험을 본 것 같습니다. 이 시험을 공부하기 위해서 산 교재는 시험 문제보다 훨씬 어려웠습니다. 데이터 베이스에 대해서 다루어본 적이 있어서 쉬울 줄 알았지만, 교재를 보면서 하나도 이해할 수 없는 것들이 마구 쏟아져 나왔습니다. 그럼에도 포기하지 않고, 이번에 시험을 같이 보는 상언님, 초원님을 끌어들여 같이 머리 싸매고 교제에 나오는 각 문제의 선택지를 해석하며 공부하였습니다. 지난 주 금요일에 2시에 수업이 끝나서 다른 사람들 집에 갈 때 우리는 8시 30분까지 남아 쉬지 않고 공부했습니다. 솔직히 “이 정도 했으면 붙어야한다.” 라는 생각도 했습니다.
  그리고 시험을 보게 되었습니다. 문제의 난이도는 교재에 비할바 못되는 아주 쉬운 문제들로 구성되어 있었습니다. 그러나 이번에도 저는 실수를 저지르고 말았습니다.
  {' '}

  회수하는 명령어를 무엇이라 하는가?”로 보았고, 답을 REVOKE라고 적었습니다. 하지만 정답은 DCL입니다.
  저는 왜 문제를 제대로 안 읽은 걸까요…? 그래도 합격에는 문제 없을 것이라고 생각합니다. 하지만 틀린
  건 너무 아쉽습니다. 반성 좀 해야겠습니다.
- 화요일은 매칭데이가 있는 날이였습니다. 그러나 저는 호감이 가는 회사가 없어서 가지 않았습니다. 게임서버개발자 과정을 듣는데 게임 회사조차 없는 취업 박람회는 어째서 가야 하는 걸까요? 그래서 저는 그 곳에 가는 대신 집에서 김장을 열심히 담갔습니다. 올해의 겨울은 낭낭하게 보낼 수 있을 것 같습니다. 그리고 다음 날을 위해 휴식을 취하였습니다. 휴식을 하면서도 독서는 놓지 않았습니다. 표도르 도스토예프스키의 ‘지하로부터의 수기’를 읽고 있습니다. 주인공의 성격이 저와 비슷한 부분이 많아서 살짝 당황스러웠습니다. 그래도 읽다보니 조금 씩 차이가 보여서 마음이 편해졌습니다. 역시 독서는 사람의 마음에 한 줄기 평화를 부여하는 것 같습니다.

- 파이썬이 갈수록 어려워집니다. 저는 목숨을 걸고 실습 문제를 푸는데, 다른 수강생 여러분들은 실습 문제를 푸는 대신에 각자 공부가 필요한 부분을 찾아서 공부하는 것 같습니다.(주로 소켓 프로그래밍에 대해서 공부하시더군요) 저는 어떻게 해야 하는 걸까요? 장기적 관점에서 볼 때는 저에게 필요한 공부를 하는 것이 맞는 것 같지만, 지금 이 순간 이 장소에서만 배울 수 있는 것을 소중히 배워나가는 것 또한 나쁘지 않은 것 같아 고민이 많아집니다. 따라서 저는 다음과 같이 행동하겠다고 결론을 내렸습니다.

  - 지금보다 더 빨리 풀고 공부할 거 한다
  - 저 짧은 시간을 이용하기 위해 라운드 로빈 스케줄링으로 공부 하되 오버헤드가 발생하지 않는 범위 내에서 시간을 최대한 단축해서 공부하는 연습을 하도록 한다

  그런데 앞으로 파이썬을 배울 날이 얼마 남지 않았습니다. 그래도 그 시간 만이라도 시간을 아껴서 공부 하는 것이 나을 것이라고 믿어 의심치 않기로 했습니다.

  아래는 제가 짠 부끄러운 코드지만 맘에 들어서 적어두겠습니다.

  ```python
  class 차 :
      리스트 = []
      def __init__(self,차주,색깔):
          self.차주=차주
          self.색깔=색깔
      def info(self):
          print('차량주인 :',self.차주)
          print('차량색깔 :',self.색깔)
          print('차종 :',self.차종)
          print('배기량 :',self.배기량)
          print('구동 :',self.구동)
          print('최고속도 :',self.최고속도)
      @classmethod
      def find(cls, 차주명):
          for i in cls.리스트:
              if i.차주 == 차주명:
                  return i
          return None
  menu = '''
  ===========
  메뉴

  A.차량등록
  B.차주검색
  ===========
  '''


  class sports(차):
      def __init__(self,차주,색깔):
          super().__init__(차주,색깔)
          self.차종 = 'sports'
          self.배기량 = '5000cc'
          self.구동 = '후륜'
          self.최고속도 = '300km/h'
          super().리스트.append(self)
  class suv(차):
      def __init__(self,차주,색깔):
          super().__init__(차주,색깔)
          self.차종 = 'sports'
          self.배기량 = '3000cc'
          self.구동 = '4륜'
          self.최고속도 = '200km/h'
          super().리스트.append(self)
  class truck(차):
      def __init__(self,차주,색깔):
          super().__init__(차주,색깔)
          self.차종 = 'sports'
          self.배기량 = '6000cc'
          self.구동 = '4륜'
          self.최고속도 = '200km/h'
          super().리스트.append(self)
  car_kind =['sports','suv','truck']

  while 1 :
      query = input(menu)
      if query == 'A' or query =='차량등록' or query == 'A.차량등록':
          차주 = input('차주 :')
          색깔 = input('색깔 :')
          if (i:=input("차종:")) in car_kind :
              a = ''
              if i=="sports":
                  a = sports(차주,색깔)
              elif i=="suv":
                  a = suv(차주,색깔)
              elif i=="truck":
                  a = truck(차주,색깔)
          else:
              print("그런 차종은 없어요")
              continue

      if query == 'B' or query =='차주검색' or query == 'B.차주검색':
              owner = input('차주 검색:')
              찾기 = 차.find(owner)
              if 찾기 :
                  찾기.info()
              else :
                  print('그런거 없음')
  ```
