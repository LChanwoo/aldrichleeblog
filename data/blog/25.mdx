## 시간복잡도

- 시간복잡도란 입력크기에 대해 어떤 알고리즘이 실행되는데 걸리는 시간이며 주요로직의 반복횟수를 중점으로 측정된다.
- 어떤 프로그램이 실행되어 종료될때까지의 시간은 OS, CPU, 램 등의 상태에 따라 달라질수도 있으므로, 시간복잡도를 설명할 때는 시간이 아니라 어떠한 알고리즘이 주어진 입력크기를 기반으로 어떠한 로직이 몇번 반복되었는가를 중점으로 설명한다.

## 빅오표기법

- 빅오표기법(Big - O notation)은 복잡도에 가장 영향을 많이 끼치는 항의 상수인자를 빼고 나머지 항을 없에서 복잡도를 나타내는 표기법
- 예) $10n^2+n$ ⇒ $O(n^2)$
- 재귀함수
  - 재귀함수의 시간 복잡도는 함수 내에서 몇번 재귀함수가 호출되는지로 결정된다
  - 예) 재귀함수를 3번씩 호출 ⇒ $O(3^2)$
- 반복문
  - 반복문이 몇겹으로 이루어졌는지로 결정된다.
  - 예) n번 반복하는 반복문 안에 n번 반복하는 반복문이 있음 ⇒ $O(n^2)$
- 계속해서 x씩 나누는 반복문
  - x의 값에 의해 결정된다.
  - 예) x가 2이면 $O(log_2n)$
    ```cpp
    int solve(n){
    	while(n>0){
    		n/=2
    	}
    	return n
    }
    ```

## 공간복잡도

- 입력 크기에 대해 어떠한 알고리즘이 실행되는데 필요한 메모리 공간의 양
- 정적변수 말고도 동적으로 재귀적인 함수로 인해 공간을 계속해서 필요로 할 경우도 포함하며, 배열, 맵, 셋 등 요소들을 담을 공간이면 다 적용된다.
- 코딩문제를 풀 때는 다음 두가지 방법을 기반으로 잡음

  1. 최대범위

     필요한 배열의 범위가 100만이면 배열의 크기를 100만으로 선언

  2. 메모리제한

     메모리제한이 512MB이면 int를 기준으로 512,000,000byte/4byte 이므로 128,000,000개의 배열 선언 가능 ⇒ 실제로는 이렇게 잡기 힘드니 1000만까지 된다고 생각하고 하면 됨
