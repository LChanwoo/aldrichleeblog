
### 들여쓰기(indentation)

- 들여쓰기는 파이썬 문법의 가장 큰 특징
  - 가독성은 높이지만, 오류가 일어나지 않도록 조심해야 함
  - 코드블록1:
    [tab]코드블록2
    코드블록3:
    [tab]코드블록4
  - 한 라인에 여러 구문이 올 경우에는 세미콜론(;)을 사용해야 함
    - a=1;b=2
- 들여쓰기는 중요하지만, 문장이 아직 안 끝난 경우에는 들여쓰기를 안해도 문법오류가 나지 않음
  - 코드
    ```python
    a=( 1 +
    			2 +
        3 +
    	      4)
    a
    ```
  - 실행결과
    ```python
    10
    ```

### 콜론(:)

- 제어구조 다음에 콜론(:)을 꼭 써야 함
- 어떤 특별한 의미가 있다기보다는 파이썬의 문법 구조
- 파이썬이 다른 언어보다 보기 쉽고 소스 코드가 간결한 이유는 바로 콜론(:)을 사용하여 들여쓰기를 하도록 만들었기 때문
- while, for, def, class문에도 역시 문장의 끝에 콜론(:)이 항상 들어감
- 콜론(:)을 빠뜨리는 경우가 많으니 특히 주의

### if 문

- if 문의 기본 구조

  ```python
  if 조건식 :
  	코드블록

  if 조건식 :
  	코드블록
  else:
  	코드블록

  if 조건식 :
  	코드블록
  elif 조건식 :
  else:
  	코드블록

  ```

- 조건식
  - 비교 연산자 사용
  - and or 연산자 활용
  - in, not in

### for 문의 기본구조

{' '}

<Image alt="" src="/static/images/python/1.png" width={800} height={800} />

- for문은 배열에 해당하는 list 요소를 다룰때 사용
- 리스트의 요소를 반복적으로 호출 처리 해야 할 때 사용함
- 리스트의 요소 전체를 호출하고 나야만 루프가 종료됨
- 조건문과 함께 사용하여 중간에 중단시킬 수 있음
- 문자열과 함께 사용 하여 문자열의 각 글자를 출력할 수도 있음
- 딕셔너리를 사용하면 딕셔너리의 키값을 모두 출력
- 숫자형 자료는 사용할 수 없음
- 예)
  - 기본
    ```python
    for a in [1,2,3]:
    	print(a)
    ```
    실행결과
    ```python
    1
    2
    3
    ```
  - range
    ```python
    li = [1,2,3]
    for a in range(3):
    	print(li[a])
    ```
    실행결과
    ```python
    1
    2
    3
    ```
  - enumerate
    ```python
    for i,e in enumerate([1,2,3]):
    	print(i,e)
    ```
    실행결과
    ```python
    0 1
    0 2
    0 3
    ```

### while문

- while 문은 특정 조건을 반복해서 수행 할 때 사용
- 가장 대표적인 반복문
- 문장 반복시 반복 조건을 검사
- 반드시 탈출 조건을 만들어 놓아야 함
  - 탈출조건을 만들지 않으면 무한 루프 발생
- 다만 일부러 무한 루프를 발생시켜야 하는 경우가 있음
  - 그럴때에도 특정한 상황에서는 반복문을 빠져 나가거나 프로그램의 실행 자체를 중단시킬 조건이 필요함
- for문과 마찬가지로 반복해서 문장을 수행 가능
- 예)
  ```python
  while True :
  	a = input()
  	if a == 'y':
  		break
  	else :
  		continue
  ```
